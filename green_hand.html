AI Chat Widget 演示

选中下方任意文字，会出现"引用到对话"按钮，点击即可将文字带入 AI 侧边栏。

功能说明

1. 拖拽悬浮球：按住右下角蓝色悬浮球可拖拽到任意位置，松开后自动吸附到左右边缘。

2. 打开对话：点击悬浮球打开侧边栏 AI 对话窗口。

3. 选中引用：在页面任意处选中文字，松开后会出现"引用到对话"提示，点击后文字会出现在输入框上方的引用框中。

4. 发送对话：输入问题后发送，引用内容会自动作为上下文附带。按 Enter 发送，Shift+Enter 换行，ESC 关闭窗口。

人工智能（Artificial Intelligence），英文缩写为 AI。它是研究、开发用于模拟、延伸和扩展人的智能的理论、方法、技术及应用系统的一门新的技术科学。

人工智能从诞生以来，理论和技术日益成熟，应用领域也不断扩大，可以设想，未来人工智能带来的科技产品，将会是人类智慧的"容器"。

接入真实 AI

当前为模拟回复。如需接入真实 API，修改初始化代码：

AIChatWidget.init({
    apiEndpoint: 'https://your-api.com/chat',
    welcomeMessage: '你好！我是 AI 助手。'
});
API 需接收 { message: "..." } 并返回 { reply: "..." }。
